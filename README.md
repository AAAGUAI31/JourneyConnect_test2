# JourneyConnect 旅行搭子小程序

一个帮助旅行者寻找志同道合旅伴（"搭子"）的社交型微信小程序。

## 功能特性

- **活动广场**：浏览所有旅行活动，支持搜索
- **活动详情**：查看活动信息、报名参加
- **发起活动**：发布新的旅行活动
- **个人主页**：管理个人信息、查看发起的/参加的活动
- **微信登录**：一键授权登录

## 技术栈

| 分类 | 技术 |
|------|------|
| 前端 | 微信小程序原生开发（WXML + WXSS + JS） |
| UI 组件 | TDesign Miniprogram |
| 后端 | 微信云开发（云函数 + 云数据库 + 云存储） |

## 快速开始

### 环境要求

- 微信开发者工具
- Node.js 14+

### 安装步骤

1. **克隆项目**
   ```bash
   git clone <repository_url>
   cd miniprogram-1
   ```

2. **安装依赖**
   ```bash
   npm install
   ```

3. **导入到微信开发者工具**
   - 打开微信开发者工具
   - 选择「导入项目」
   - 选择本项目目录
   - 填写 AppID

4. **开启云开发**
   - 在微信开发者工具中点击「云开发」
   - 创建云环境（记录环境 ID）
   - 在 `app.js` 中更新 `env` 为你的环境 ID

5. **创建数据库集合**
   - 进入云开发控制台 → 数据库
   - 创建以下集合：
     - `users`
     - `activities`
     - `activity_members`

6. **部署云函数**
   - 展开 `cloudfunctions` 目录
   - 右键 `login` → 「上传并部署」
   - 右键 `user` → 「上传并部署」

7. **编译运行**
   - 在微信开发者工具中点击「编译」
   - 即可在模拟器中预览

详细说明请参考 [开发指南](docs/guide.md)。

## 页面预览

### 首页（活动广场）
- 展示所有旅行活动卡片
- 支持关键词搜索（标题/地点）
- 点击卡片进入活动详情

### 活动详情
- 显示活动信息（标题、时间、地点、描述）
- 展示已报名人员
- 报名/取消报名

### 创建活动
- 填写活动信息（标题、时间、地点等）
- 选择旅行风格标签
- 上传封面图片
- 发布后跳转详情页

### 个人主页
- 微信一键登录
- 编辑个人资料
- 查看「我发起的活动」「我参加的活动」
